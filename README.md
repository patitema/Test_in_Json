# Сайт для создания тестов на Flask

Проект представляет собой веб-приложение на **Flask**, предназначенное для создания и управления тестами. С помощью этого приложения пользователи могут:

- Создавать тесты через удобный интерфейс админки.
- Импортировать тесты из JSON файлов (пример таких файлов можно найти в папке `data`).
- Управлять тестами, редактировать и удалять их через административный интерфейс.

## Функционал:
- **Админка:** Интерфейс для управления тестами, который позволяет добавлять, редактировать и удалять тесты.
- **Импорт тестов:** Поддержка импорта тестов из JSON файлов для упрощения работы с большими объемами данных.

# Инструкция по запуску проекта

## Требования:

- Python 3.7+
- pip

## Шаги для запуска:

1. **Клонировать репозиторий:**

   Скачай проект на свой компьютер, используя команду:

   ```bash
   git clone https://github.com/patitema/Test_in_Json.git
   cd Test_in_json
   ```

2. **Создать виртуальное окружение:**

   Для создания изолированного окружения и установки зависимостей, нужно создать виртуальную среду. Для этого выполни следующие шаги:

   - На Windows:

     ```bash
     python -m venv venv
     ```

   - На Linux/macOS:
     ```bash
     python3 -m venv venv
     ```

3. **Активировать виртуальное окружение:**

   - На Windows:

     ```bash
     .\venv\Scripts\activate
     ```

   - На Linux/macOS:
     ```bash
     source venv/bin/activate
     ```

   После активации виртуального окружения, в командной строке появится префикс `(venv)`.

4. **Установить зависимости:**

   Для установки всех необходимых библиотек из файла `requirements.txt`, используем следующую команду:

   ```bash
   pip install -r requirements.txt
   ```

5. **Запуск проекта:**

   Теперь, когда все зависимости установлены, можно запустить сервер Flask. Для этого в корне проекта выполните команду:

   ```bash
   python app.py run
   ```

   Сервер запустится по умолчанию на порту 5000, и ты сможешь открыть проект в браузере по адресу http://127.0.0.1:5000.

6. **Интерфейс админки:**

   Для доступа к админке нужно зайти под админ пользователем. По умолчанию это: 
   ```
   username:admin 
   pass:adm1n
   ```

7. **Импорт тестов из JSON:**

   В папке `data` проекта ты можешь найти пример JSON файла с тестами. Для импорта тестов, используй соответствующий функционал на странице админки или с помощью API, если такой предусмотрен.
